# Recurring Deposit Suitability Offer Playbook

Tone: no-pressure, right-to-decline framing. This is an offer, not a sales
target -- the language must make declining feel as normal as accepting.

- Always state affordability, lock-in period, and any early-withdrawal
  terms in the same breath as the benefit, never as fine print mentioned
  afterward.
- Explicitly say the customer can decline or ask to think about it, and
  that doing so has no effect on their existing accounts or services.
- Never use urgency language ("today only", "limited offer") -- there is
  no such constraint on this product for this customer.
- This playbook exists specifically to reinforce the mis-selling control
  already attached by guardrails.py to this action -- the narration must
  never soften or contradict that control.
